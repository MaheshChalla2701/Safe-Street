# SafeStreet Mobile App

This directory contains the mobile application for **SafeStreet**, an AI-based system designed to detect various types of road damage from real-world images. The mobile app allows users to capture images, visualize road damage detection results, and receive alerts.

It is built using **React Native** and **Expo**.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npx expo start
   ```

From there, you can open the app in a development build, Android emulator, iOS simulator, or the Expo Go app.

## Project Structure
- `app/` - Application screens and file-based routing
- `components/` - Reusable UI components
- `assets/` - Static assets like images and fonts
- `hooks/` - Custom React hooks
- `constants/` - Configurations and constants
