# SafeStreet Web Admin Panel

This directory contains the web-based admin dashboard for **SafeStreet**, an AI-powered system designed to detect and alert road damage. The admin panel allows civic bodies and administrators to monitor reported road conditions, view classifications (Crack, Pothole, Patch, Surface), and track system alerts.

It is built using **React.js**.

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm start
   ```

This runs the app in development mode. Open [http://localhost:3000](http://localhost:3000) to view it in your browser. The page will reload when you make changes.

## Scripts

### `npm test`
Launches the test runner in interactive watch mode.

### `npm run build`
Builds the app for production to the `build` folder. It minifies the build and optimizes for the best performance.
